# Site Info Scout

Site Info Scout is a lightweight WordPress admin plugin that generates a support-ready site report with environment details, active plugins, theme information, and safe diagnostic health flags.

## Features

- Environment summary
- Active plugins inventory
- Active theme details
- Health flags for common support issues
- Copyable support report
- TXT export
- CSV export
- Admin-only and read-only by design

## Plugin Identity

- **Plugin Name:** Site Info Scout
- **Slug:** `site-info-scout`
- **Text Domain:** `site-info-scout`
- **Namespace:** `Zignites\SiteInfoScout`
- **Prefix:** `zigsiteinfoscout_`

## Goals

- Make support and troubleshooting faster
- Provide a clean WordPress-native diagnostic summary
- Stay lightweight and WordPress.org friendly
- Avoid remote calls, telemetry, and risky write operations

## Development Notes

This plugin is being developed for WordPress.org distribution and follows WordPress coding standards, internationalization practices, and review-friendly architecture.
